HTML Cheat Sheet

Content Sectioning

Tag Name Description
<address> Address Indicates enclosed HTML provides contact information
<article> Article Represents self-contained content intended for individual distribution (e.g. blog entry, newspaper article, user-submitted comment)
<aside> Aside Represents content that is only indirectly related to main content
<footer> Footer A footer typically contains informatoin about the author of that section, copyright data, or links to related docs
<header> header Represents introductory content, like navigational aids or logos
<h1>, <h2>...<h6> Sub-headers Represents levels of section or sub-headings
<main> Main Represents the dominant content of <body>
<nav> Navigation Navigation links, either to other pages, or to sections within the page
<section> Section Represents a standalone section that doesn't have a more specific semantic element to represent it.

Table Tags

Tag Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<td> Table Data The table data container.
<tfoot> Table Foot The set of rows defining the footer in a table.

Table Attributes

Attribute Name Description
colspan Column Span Defines how many columns a td element should span.
rowpsan Row Span Defines how many rows a td element should span.

Text Content

Tag Name Description
<blockquote> Block Quote Indicates that enclosed HTML is an extended quote
<div> Division Generic container for flow content, usually for CSS styling purposes
<dl> Description List Encloses a list of groups of terms (specified using <dt>) and descriptions (<dd>)
<dt> Description Term Specifies a term in a description or definition list. Must be used inside <dl> and is usually followed by <dd>
<dd> Description Details Provides description, definition, or value for preceding term <dt> in a description list <dl>
<figure> Figure Represents self-contained content with optional caption. Figure and caption are referenced as single unit
<figcaption> Figure Caption Caption for its parent <figure>
<ol> Ordered List Encloses a numerical list
<ul> Unordered List Encloses a bulleted list
<li> List Item Denotes an item in an <ol> or <ul>
<p> Paragraph Represents a paragraph
<hr> Horizontal Rule Represents a thematic break between paragraph-level elements

Image and Multimedia

Tag Name Description
<area> Area Defines an area inside an image map that has predefined clickable areas
<audio> Audio Embeds sound content. May contain one or more audio sources using src attribute.
<img> Image Embeds an image into the document.
<map> Map Used with <area> area elements to define an image map.
<track> Track Used as a child of the media elements <audio> and <video>. It lets you specify timed text tracks, for example to automatically handle subtitles.
<video> Video Embeds a media player which supports video playback into the document.
